LAN-SHARE 使用教程

  1. 功能 & 亮点
  2. 部署教程
  3. 基本的配置
  4. 其他

LAN-Share 是一个基于 Node.js 的用于局域网内文件分享的工具，易于配置与部署。
项目地址：https://github.com/songquanpeng/lan-share，欢迎大家 Star！

功能 & 亮点

  1. 首页文章自动根据浏览量和创建时间排序
  2. 借助于 sm.ms 图床直接

部署教程

  1. 第一步,我们需要安装 Node.js v10.14.2 或兼容的版本，请从官网下载
  2. 第二步，使用 Git 将项目 克隆下来，或者直接下载
  3. 第三步，解压缩，并打开此文件夹，按住 Shift 键然后单击鼠标右键，选择在此打开 Power Shell
  4. 第四步，打开终端后，输入 npm i 安装依赖
  5. 第五步，依赖安装完成后，输入 npm start 启动服务器
  6. 现在打开http://localhost:3000/ ，你就可以开始在局域网内分享文件了！

基本的配置

打开位于项目根目录下的 config.js，在里面你可以配置端口号和 token，拥有 token 的用户可以删除文件。

其他

  1. 用户上传的文件位于项目根目录下的 /public/downloads/ 文件夹内
  2. Windows 下你可以使用 ipconfig 命令查询自己的 ip 地址
