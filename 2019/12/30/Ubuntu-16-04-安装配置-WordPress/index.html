安装与配置 MySQL

安装

参考：https://www.digitalocean.com/community/tutorials/how-to-install-mysql-on-ubuntu-16-04

配置

输入 sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf 进行配置
运行远程访问，注释掉 bind-address = 127.0.0.1

创建用户以及赋权
mysql -uroot -p

CREATE USER ‘username‘@’%’ IDENTIFIED BY ‘password’;
GRANT ALL PRIVILEGES ON . TO ‘username‘@’%’;
FLUSH PRIVILEGES;

GRANT ALL PRIVILEGES ON . TO ‘song‘@’%’ WITH GRANT OPTION;

https://www.cnblogs.com/cnblogsfans/archive/2009/09/21/1570942.html

GRANT ALL PRIVILEGES ON blog.* TO “wordpress”@”localhost” IDENTIFIED BY “Word12!@”;
